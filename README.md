# 🎬 Photomotion

Photomotion is a lightweight web application that transforms **3–5 photos** into a cinematic slideshow video with smooth **dissolve** and **slide** transitions, accompanied by a default background music track. Designed with a clean and intuitive interface, it enables users to quickly generate memorable video montages directly in their browser.

🌐 **Live Demo:** https://photomotion.vercel.app/

📂 **GitHub Repository:** https://github.com/bhavukm007/Photomotion

---

## ✨ Features

- 📷 Upload **3–5 images** from your device
- 🎞️ Automatic slideshow generation
- 🌊 Smooth **dissolve** and **slide** transition effects
- 🎵 Default background music integration
- 👀 Live preview before export
- 📤 Export the final slideshow as a video
- 📱 Responsive and user-friendly interface

---

## 🎯 How It Works

1. Upload **3–5 photos** from your device.
2. The application validates the selected images.
3. Photos are arranged into a slideshow sequence.
4. Smooth transition effects are applied.
5. A default background music track is added.
6. Preview the slideshow.
7. Export the final video.

---

## 📋 Requirements

- Upload between **3 and 5 images**
- Apply smooth slideshow transitions
- Include default background music
- Generate an exportable video
